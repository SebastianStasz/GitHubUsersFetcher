# GitHubUsersFetcher

### About

A simple application that retrieves github users data and displays: the image, name and number of repositories for each user.

### Features

- Custom AsyncImage fetcher
- Image Cache
- Infinite Scrolling
- Animated placeholder for image

### Technologies

- Combine
- UIKit
